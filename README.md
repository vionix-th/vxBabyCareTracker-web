# Baby Care Tracker

Shared, real-time baby care tracking for parents and caregivers. Log bottle feeds, breastfeeding, diapers, sleep, crying, pumping, weight, and notes in a shared family space.

**Live app → [vxBabyCareTracker.web.app](https://vxBabyCareTracker.web.app)**

## Features

- Shared family spaces — invite co-parents and caregivers via link or QR code; everything syncs in real time
- Fast logging — quick-entry panels for all care types; large tap targets; progressive disclosure of optional fields
- Feeding sessions — bottle, breast, top-ups, and mixed feeds grouped into sessions
- Dashboard summaries — today and rolling 24h views of intake, wet diapers, stools, sleep, and crying
- Reports and exports — daily trends, doctor report, printable PDF, clinical CSV
- JSON backup and restore — versioned, component-selected backup with merge or full replacement restore
- Role-based access — admin, parent, caregiver, and view-only roles
- Localization — English and Thai with per-account language preference
- PWA — installable, works offline for previously loaded data

## Tech Stack

Vite, React, TypeScript, Firebase Auth, Cloud Firestore.
